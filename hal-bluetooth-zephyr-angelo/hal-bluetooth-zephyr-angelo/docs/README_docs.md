# Documentação e relatório

Esta pasta (`docs/`) serve para relacionar este projeto de código com o relatório teórico produzido em LaTeX (Overleaf).

Sugestão de organização:
- Colocar aqui uma cópia em PDF do relatório final sobre o HAL + Bluetooth + Zephyr.
- Opcionalmente, incluir o arquivo `.tex` exportado do Overleaf.

No relatório, são descritos:
- Os objetivos do projeto.
- A configuração do ambiente Zephyr.
- Os testes com exemplos básicos (`hello_world`, `blinky`).
- A migração para a placa SAMD21 Xplained Pro.
- As tentativas de integração com o módulo Bluetooth (BTLC1000).
- Os erros encontrados, soluções parciais e aprendizados.

Este repositório de código, junto com o histórico de commits no GitHub,
compõe a parte prática/individual exigida pelo professor (60% da nota).
